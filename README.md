# Python_Piscine

A structured journey through Python – from absolute basics to advanced concepts like decorators, environments, and Pydantic.  
This repository contains 11 independent modules, each focusing on a specific set of skills. Completed as part of a self‑directed Python course.

## 📚 Modules Overview

| Module | Topics Covered |
|--------|----------------|
| **00** | Basics: `print`, `if`, `while`, loops, simple statements |
| **01** | Functions: defining, calling, parameters, return values |
| **02** | Exception handling (`try`/`except`), OOP: classes, methods, attributes |
| **03** | External arguments (`sys.argv`), built‑ins: `min`, `max`, `round`, `input`, `print`, `sum`, `len`, etc. |
| **04** | File I/O: reading, writing, editing files, handling open errors |
| **05** | Small project: build a moderately complex project using classes and methods |
| **06** | Code organisation: importing from other files, modular programming |
| **07** | Game development: a small game with complex logic, OOP, and state management |
| **08** | Environments: virtual environments, `.env` files, keeping secrets out of code |
| **09** | Pydantic: data validation, enforcing argument constraints |
| **10** | Advanced: `lambda`, decorators (custom and built‑in: `lru_cache`, `singledispatch`, `partial`, `reduce`) |

## 🚀 Getting Started

Each module is independent. Navigate into a module’s folder and run the Python files as described in its internal instructions (if any).

### Prerequisites

- Python 3.10 or later
- Some modules may require additional packages (e.g., `pydantic`, `python-dotenv`). Check each module’s `requirements.txt` or comments.

## 📖 Learning Outcomes

After completing these modules, you will be able to:

- Write idiomatic Python from scratch
- Structure programs using functions, classes, and modules
- Handle errors gracefully
- Work with files and system arguments
- Build a complete small game with OOP
- Use virtual environments and environment variables
- Validate data with Pydantic
- Apply functional programming techniques (lambda, decorators)
